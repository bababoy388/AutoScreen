==================================================================
Секция [Telegram]
==================================================================
token         = токен бота Telegram
chat_id       = ID чата или канала
proxy         = (необязательно) HTTP/HTTPS-прокси, например http://user:pass@host:port
                Если не нужен – оставьте пустым

==================================================================
Секция [Schedule]
==================================================================
mode          = режим работы:
                once      – выполнить один раз и выйти

                     – запускать ежедневно в указанное время
                interval  – повторять через заданные интервалы (в минутах)

time          = время запуска в формате ЧЧ:ММ (используется только для mode=daily)
interval_minutes = интервал повтора в минутах (используется только для mode=interval)

==================================================================
Секции [Plot_*]
==================================================================
Каждая секция описывает один график. Имена секций должны начинаться с "Plot_".
Параметры:

millUuid       = UUID мельницы (строка)
from_minutes   = смещение начала диапазона данных в минутах от текущего момента
                 (например, -60 означает "60 минут назад")
to_minutes     = смещение конца диапазона (обычно 0 – "сейчас")
                 Если нужно абсолютное время, используйте параметры from_time / to_time
                 в формате "YYYY-MM-DDTHH:MM:SS.fffZ" (одновременно с минутами не задавать)

info_host      = IP или хост сервера информации (без http://)
info_port      = порт для запроса имени CSV-архива
download_host  = IP или хост сервера скачивания
download_port  = порт для скачивания архива

columns        = имена колонок через запятую (без пробелов или с пробелами – не важно,
                 пробелы удаляются автоматически). Все колонки должны присутствовать в данных.
title          = заголовок графика
ylabel         = подпись оси Y
kind           = тип графика: line, bar, scatter, ... (см. документацию pandas.DataFrame.plot)
figsize        = размер графика в дюймах: ширина,высота (например 12,4)
grid           = показывать сетку (True / False)
save           = имя файла для сохранения графика (будет помещён в папку graphs рядом со скриптом)
